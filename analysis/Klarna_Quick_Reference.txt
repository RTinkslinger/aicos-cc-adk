KLARNA CEO ANALYSIS - QUICK REFERENCE
=====================================

VIDEO: "SaaS is Dead: Why Systems of Record Will Die in an Agentic World"
GUEST: Sebastian Jurisch (Klarna CEO)
SHOW: 20VC with Harry Stebbings
DURATION: 89 minutes

CORE THESIS IN ONE SENTENCE:
Software creation cost → zero kills SaaS economics, but agents enabling one-click data migration kills the lock-in moat.

THREE KEY CLAIMS:
1. Software creation cost collapsing to zero
   - Everyone will be able to generate software at any time
   - No defensibility from code speed/quality

2. Agents collapse data switching costs
   - "How do I get all my data from the existing vendor and move it to the new vendor with the help of AI through one click?"
   - This breaks the primary SaaS moat

3. SaaS multiples compress from 20-30x to 1-2x revenue
   - Already down to 5-10x
   - Market "woke up" to this recently
   - Comparison: Chegg at 0.2x after ChatGPT

KLARNA AS PROOF POINT:
- 7,000 → 3,000 employees (50% reduction)
- No new capital required
- 600 agent jobs automated in customer service
- Built own AI-first system (couldn't use off-shelf SaaS)
- Projected: <2,000 employees by 2030

"COMPANY IN A BOX" FUTURE ARCHITECTURE:
- Open source accounting + open source CRM + AI agent on top
- Agent handles all interaction ("bookkeep this invoice", "set up customer account")
- No traditional SaaS UI
- Unified data context for AI

DIRECT PORTFOLIO IMPACTS:
HIGHEST THREAT: Unifize (manufacturing QA vertical SaaS)
  → This is the exact use case Jurisch describes
  → Pure process automation = agent-replaceable
  → P0 priority: Deep dive with founders

HIGH THREAT: Highperformr (social media management SaaS)
  → Vertical SaaS with low data switching cost
  → P1 priority: Assess if pivoting to agent infrastructure

MEDIUM-HIGH THREAT: CodeAnt (dev tools)
  → Dev tools have longer runway
  → Survives if becomes agent-integrated vs. siloed SaaS
  → P1 priority: Verify integration roadmap

LOWER THREAT: Cybrilla, Dodo Payments, Smallest AI (infrastructure)
  → Infrastructure more defensible than SaaS UI layer
  → Can be orchestrated by agents without replacing core value

ALIGNED: Confido Health (building agents, not SaaS)
  → Already the architecture Jurisch describes
  → Moat is regulatory, not software

THESIS IMPACT:
Validates: Agentic AI Infrastructure thesis
  → Harness (orchestration layer) is the winner
  → BUT suggests consolidation > fragmentation
  → One broad AI system > many specialized agents

New Thread: SaaS Death / Agentic Replacement
  → Not theoretical; happening now (Klarna)
  → Timeline: 3-5 years, not 2030
