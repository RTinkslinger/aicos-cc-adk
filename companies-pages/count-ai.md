### Product Overview

- Computer vision-based quality control system for textile knitting machines

- System includes hardware box and user interface installed on each knitting machine

- Detects fabric defects in real-time during production process

- Reduces fabric wastage from ~1.5% to 0.1% through early detection

- Currently covers 80% of common fabric types

- False positive rate: 1-2 per day per machine

- Each unit has 2 cameras for different angle detection
