- Data management module:

  - Segregates data into folders

- Data doesn’t train Lucio’s model, ensuring no impact on the system

- Lucio doesn’t hallucinate information, unlike early AI models

  - Other tools (redaction, redline, OCR) have longer processing times than regular PDF tools

## Euclid’s (UK Law Firm) Partners Feedback 2025-08-15 (+ve):

Out of interest, here is an email I sent to the team yesterday based on my initial experience:

"I wanted to share some initial feedback to help encourage everyone to start using the platform.  In my view, it is <u>really</u> good at doing certain things, for example:

- I asked it (via the Assistant function) to compare to versions of a complex document and summarise the changes. The report was super helpful and avoided the need to look through all the edits.

- I asked it (via the Assistant function) to compare the same long document with a briefing paper to spot inconsistencies. Again, the report was very helpful.

- I used the Briefcase function to query a case file to find the name of a trade association I couldn’t remember. It found it quickly.

- I used the Briefcase function to find a spreadsheet listing customers, which I knew was attached to an email but didn’t know the date. It worked.

- I used the Briefcase function to create a summary of all interactions with the CMA on the Emma Sleep case on a certain point. It worked.

- I used the Chronologies function to create a detailed chronology of the Emma case. It produced a 500 page table of all dates, including ones referred to in other documents. As this was a bit long, I then asked Assistant to summarise the information in that table. It worked.

As we learned in the training, the Briefcase function works by analysing documents uploaded to a project.  The good news is that it is easy to upload documents from Net Docs to a project, <u>as long as they are saved in Net Docs to start with</u>.  You also need to have ndSync installed.  I’m happy to show anyone how to do this.  The tool coped seamlessly with different formats, including emails and PDF scans of Egress email exchanges with the CMA.  The less good news is that, since Lucio does not (yet) offer full integration with Net Docs, you can’t interrogate the matter folder on Net Docs directly, so will need to periodically upload new documents on an ongoing matter before you can query them.”
