Sharing the note for our $50K investment in Potpie as part of their seed round of $2M a $12.5M valuation cap, led by Emergent VC - $1.5M.
About the Company
Potpie is building an AI-driven behavioural code auditor that enables developers to ship bug-free code to production confidently by automatically validating the intended code behaviour at every git push.
Problem Statement: engineers today spend nearly 5x the time testing vs writing code, with ~40% of total developer bandwidth going into writing and maintaining test cases. Despite this, $2.4T in losses were attributed to poor-quality software shipped in the US in 2022. Momentum’s product uses code behaviour analysis, test generation, and automated validation to ensure code correctness in real-time.
Core Product Features:
  - Behavioural code auditor that analyzes code functionality, efficacy, and inter-compatibility.
  - Automatically generates end-to-end tests and integrates them into the repo.
  - Visualizes codebase architecture to catch dependency issues.
  - Ensures every commit meets defined behavioural criteria before production deployment.
Traction & Progress
  - Open-source v1 launched (FY25 Q1).
  - 15+ pilots landed in the past 8 weeks with companies like Tata 1mg and Bain & Co.
  - 2 pilots completed, 2 ongoing, and 15 in pipeline.
  - Product roadmap includes enterprise rollout with self-healing tests, debugging in natural language, SOC2/GDPR compliance, and CI/CD integrations.
Founders
[Aditi Kothari](https://www.linkedin.com/in/aditi-kothari/) (CEO) – Previously Product @ Razorpay and Bounce.
- Built a product line at Bounce saving $660K annually.
- Drove one of Razorpay’s fastest-growing GTM channels from 0→1.
[Dhiren Mathur](https://www.linkedin.com/in/dhirenmathur/) (CTO) – Previously Engineering @ Cisco Webex, vidyo.ai.
- Led backend engineering at vidyo.ai; scaled AI/video workloads from 0→$1M ARR.
- Built global compliance products at Webex during pandemic surge.
Investment Considerations
- Massive market gap: AI code generation (Copilot, Replit, Cody) has outpaced the ability to test and validate — creating a $10B+ opportunity in automated software reliability.
- Behavioural testing moat: Focuses on intent validation instead of syntax-based QA, positioning Momentum as the “auditor for code behaviour.”
- Enterprise-first roadmap: Early pilots with large tech-first companies (Tata 1mg, Bain) validate market pull.
- Founders deeply technical: Strong GTM + infra backgrounds across high-scale systems (Razorpay, Webex, Bounce).
Investment Risks
- Early stage of adoption — open-source model will need clear monetization path.
- Competes with established testing automation and DevOps suites (e.g., Testim, Mabl, Harness).
- Requires tight product-market alignment between AI code generation and enterprise DevOps workflows.
