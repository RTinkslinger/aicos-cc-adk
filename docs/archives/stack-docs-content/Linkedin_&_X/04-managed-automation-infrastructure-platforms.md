# Managed Automation Infrastructure Platforms (Apify-class Systems)

## Overview

These platforms provide managed compute infrastructure for running
browser automation scripts ("Actors").

## Core Architecture

-   Hosted container execution
-   Actor-based packaging
-   Dataset and storage primitives
-   Proxy add-ons
-   MCP server exposure

## Sub-Options

-   Public and private Actors
-   MCP server integration
-   Proxy pools (datacenter/residential)
-   Webhook-triggered execution

## Capabilities

-   Custom LinkedIn/X scraping
-   Headless automation via Playwright/Puppeteer
-   Scheduled runs
-   Dataset exports

## Infrastructure Features

-   Compute-unit billing
-   Parallel run limits
-   Persistent key-value stores
-   Logs and metadata tracking

## Structural Gaps

-   Maintenance burden for UI changes
-   Compute cost variability
-   CAPTCHA and auth friction
