## L49-52: New Query Functions + Sparkline Data

**Goal:** Backend support for person-specific data and dashboard sparklines.

**New query functions added to `src/lib/supabase/queries.ts`:**
- `fetchPersonObligations(personId)` — obligations filtered by person_id
- `fetchPersonInteractions(personName, limit)` — interactions via participants array contains
- `fetchPersonObligationStats(personId)` — iOwe/theyOwe/pending/fulfilled counts per person

- `fetchInteractionCountsByWeek(weeks)` — weekly interaction counts for sparkline
- `fetchObligationResolutionByWeek(weeks)` — weekly created vs resolved for dual sparkline
- `getISOWeek(date)` — ISO week string helper for bucketing
