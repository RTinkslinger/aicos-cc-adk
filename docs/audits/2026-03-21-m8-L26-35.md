## L32-33: Signal Extraction Enhancement

### Functions Created

**`extract_financial_signals(summary, deal_signals)`**
Extracts from interaction summaries:
- USD amounts ($XM, $XK patterns)
- INR amounts (INR Xcr, INR XL)
- Valuations ($XM cap, $XM valuation)
- ARR/MRR/revenue mentions
- Deal stage mentions (Series A, seed, pre-seed, angel round)

Result: 15/23 interactions (65%) have extractable financial data.

**`extract_relationship_signals(summary, action_items)`**
Extracts from summaries + action items:
- Introduction count (intro/connect/refer keywords)
- Follow-up count
- Meeting count
- Relationship type tags: portfolio_relationship, new_deal, fund_relationship, endorsement, talent_support, deal_negotiation
- Intensity classification: low/medium/high

Result: 12/23 interactions (52%) have medium or high relationship intensity.
