# Claude.ai Memory Entry #10: Deep Research Protocol

Add this as a new memory entry in Claude.ai Settings -> Memory:

---

When Aakash says "research deep and wide" or asks for exhaustive/comprehensive/thorough research: Run 6-10 parallel web searches from different angles (market structure, key players, recent funding, technology, competitive dynamics, contrarian views). Synthesize into a structured report: Executive Summary, Key Findings, Market Map, Open Questions. Always end by flagging connections to active thesis threads and offering to write new threads to the Notion Thesis Tracker (data source 3c8d1a34-e723-4fb1-be28-727777c22ec6).

---

**Character count: ~494** (under 500 limit)

## Why This Works

Claude.ai doesn't have Parallel MCP tools, but it DOES have:
- WebSearch (multiple parallel searches)
- WebFetch (deep dive into specific URLs)
- Notion MCP tools (can write to Thesis Tracker)

So the Claude.ai version manually replicates the parallel research pattern:
1. Decompose the topic into 6-10 search angles
2. Run searches (Claude.ai can do multiple in one turn)
3. Synthesize into structured output
4. Flag thesis connections + offer to sync to Notion

It won't be as deep as Parallel AI's engine, but it's the best available on that surface — and critically, it maintains the same trigger phrase ("research deep and wide") and the same thesis sync behavior.
