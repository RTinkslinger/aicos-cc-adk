## Key Concepts Extracted

### Architecture
- 11 Claude Code subagents, each with unique analytical method and declared blind spots
- 6 deliberate polarity pairs (structural opponents prevent groupthink)
- Coordinator agent routes and enforces protocol
- Model split: Opus for depth-heavy lenses, Sonnet for speed-critical

### Polarity Pairs
1. Socrates vs Feynman — top-down destruction vs bottom-up rebuild
2. Aristotle vs Lao Tzu — classify everything vs categories are the problem
3. Sun Tzu vs Aurelius — win external game vs govern internal one
4. Ada vs Machiavelli — formal purity vs messy human incentives
5. Torvalds vs Watts — ship concrete vs question existence of problem
6. Musashi vs Torvalds — wait for perfect moment vs ship now

### 3-Round Protocol
1. **Independent Analysis** — 400 words max, parallel execution
2. **Cross-Examination** — 300 words, must engage 2+ others by name, sequential
3. **Crystallization** — 100 words max, no new arguments

### Anti-Recursion Safeguards
- Hemlock Rule: Socrates can't re-ask addressed questions
- 3-level depth limit on questioning
- 2-message exchange cutoff between any pair

### Decision Rules
- 2/3 majority = consensus (dissent in Minority Report)
- No majority = present dilemma, don't force consensus
- Domain expert gets 1.5x weight

### Pre-built Triads
11 domain-specific 3-member groups for: architecture, strategy, ethics, debugging, innovation, conflict, complexity, risk, shipping, product, founder
