## 2. Product Portfolio & Technology Stack

Kimatica’s product suite targets the end-to-end radio-planning workflow, offering tools that automate complex network design tasks.

### 2.1 iPlanner – Indoor Coverage Engine
iPlanner is an indoor coverage planning tool designed to help engineers electronically design, configure, and optimize in-building cellular networks [6]. It features an integrated RF component library and provides one-click link budget analysis that automatically calculates required RF power along the link [6]. 

### 2.2 ACiD – Automatic Code Identifier
ACiD automates the allocation of cell frequencies and codes across GSM, UMTS, LTE, and 5G networks [4]. It utilizes a single optimization algorithm to handle multiple technologies and generates statistical histograms to assist engineers in refining code allocations [4].

### 2.3 CellViewer – 3D GIS Visualiser
CellViewer is an outdoor visualization tool based on OpenGL architecture that renders physical environments and network infrastructure in both 2D and 3D [7]. It allows engineers to validate outdoor site locations against real-world GIS data (vector clutter, raster clutter, and DTM data) [7].
